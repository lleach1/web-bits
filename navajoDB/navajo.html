Instructions:

You will need to install the font Aboriginal Serif for the special character required by the Navajo language to display. You can download a copy here. On a Widnows machine you can drag the file absechr7.ttf to C:\WINDOWS\Fonts or use the Font Control Panel and choose Install New Font on the File menu.

Supported Browsers:

  • Windows: Firefox 0.8, Mozilla 1.6 or Internet Explorer 6
  • Macintosh: Firefox 0.8 or Mozilla 1.6

Use the buttons below to enter the special characters and your keyboard to enter any other characters.
